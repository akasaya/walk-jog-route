# Project Overview

## Project name

Walk Jog Route

## Goal

現在地から、その日の散歩またはジョギングコースを自動提案する Android / Web アプリを作る。

ユーザーは現在地、移動モード、目標距離または時間、気分や条件を指定し、アプリは複数のコース候補を返す。

## Main use cases

- 現在地から散歩コースを提案する
- 現在地からジョギングコースを提案する
- 距離や時間に合う周回コースを提案する
- 提案理由を表示する
- お気に入りコースや履歴を保存する

## Learning goals

このプロジェクトはアプリ開発だけでなく、以下の練習を目的とする。

- Android / Web アプリ開発
- backend API 開発
- ルート生成ロジック
- テスト設計
- ハーネスエンジニアリング
- Docker
- GitHub Actions
- AWS デプロイ
- CI/CD
- Infrastructure as Code
- AI エージェントを使った開発運用
